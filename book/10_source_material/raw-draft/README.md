# Source Material — Raw Draft

Place your existing draft files here. Supported formats:
- Markdown (`.md`)
- Plain text (`.txt`)
- Word documents (`.docx`) — convert to Markdown before processing

## File Naming

Use descriptive names: `chapter-1-rough.md`, `intro-ideas.md`, `old-manuscript-v3.md`
